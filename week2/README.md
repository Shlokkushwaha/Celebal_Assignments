# SQL Superstore Data Analysis & ShopEase E-Commerce Database

## Project Overview

This project demonstrates SQL-based data analysis and database management using MySQL. It consists of two parts:

1. **Superstore Sales Data Analysis** – Analyzing sales data using SQL queries to generate business insights.
2. **ShopEase E-Commerce Database Assignment** – Designing and querying a relational database containing customers, products, orders, and order items.

Analyze sales data using SQL with:
